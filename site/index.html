Data Sources

Seven sources. One morning message. Zero config changes.

🌤️

Weather

Current conditions, today's high/low, rain chance, and feels-like — from Open-Meteo (free, no key).

📅

Calendar

Today's events, busy-hours summary, and location info — via Google Calendar API.

🐙

GitHub

Assigned issues, review-requested PRs, and recently active repos — via the gh CLI.

🚆

Bahn

Next departures at your station with delay info — via the free DB transport.rest API.

🔴

Reddit

Top posts from your subreddits — via public RSS feeds, zero auth needed.

📰

News

Headlines from your RSS feeds — deduplicated across sources via feedparser.

📬

Email

Unread count and recent subjects — via stdlib imaplib, zero external deps.

How It Works

01

Parallel Fetch

All 7 sources are fetched concurrently via ThreadPoolExecutor. One slow source never blocks the rest.

→
02

LLM Summarizes

A structured prompt is built from all source data plus yesterday's comparison — then fed to any LLM you choose.

→
03

WhatsApp Delivers

Hermes cron sends the compact briefing message to your phone. Read it in 5 seconds, know everything.

Start your mornings informed.
